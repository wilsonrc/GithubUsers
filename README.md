### Actual features:
+ List of github users with infinite scroll.
+ Save favorite github users.
   
### App Architecture:
+ Kotlin
+ MVVM and LiveData (Architecture Components)
+ Dependency Injection (Dagger 2)
+ Rxjava
+ Room
+ Package by feature
